About the Site

This site is built with simplicity and speed in mind. It's a static website powered by:

  • Go + SQLite for data management
  • Bootstrap for responsive design
  • AWS S3 + CloudFront for fast, reliable hosting

All event data is carefully curated and regularly updated to ensure accuracy.

Infrastructure & Reliability

We take site availability seriously. Our infrastructure is designed for high availability:

  • Global CDN: CloudFront delivers content from edge locations worldwide for fast load times
  • Multi-Region Redundancy: Automatic failover between US East and US West regions
  • 99.9% Uptime: If one AWS region goes down, the site automatically switches to the backup region in seconds
  • HTTPS Everywhere: Secure connections with auto-renewing SSL certificates
  • Open Source: All code is available on GitHub for transparency and community contributions

Built with enterprise-grade AWS infrastructure to ensure you can always access event information when you need it.
